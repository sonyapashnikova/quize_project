# Интерактивная Викторина на эрудицию

Этот проект представляет собой простую интерактивную викторину на разные темы, реализованную с использованием HTML, CSS и JavaScript.

## Технические требования

- **HTML:** Структура викторины, включая вопросы (минимум 10), варианты ответов (кнопки), прогресс-бар, кнопку «Результаты» и экран результатов.
- **CSS:** Визуальная индикация правильных/неправильных ответов (зелёный/красный), стили прогресс-бара, стиль оформления вопроса.
- **JavaScript:**
    - Хранение вопросов в массиве.
    - Логика отображения текущего вопроса и подсчета баллов.
    - Обработка выбора, подсветка ответов.
    - Переход к следующему вопросу.
    - Отображение результата по окончании викторины.
    - Кнопка "Начать заново".

## Стек

- HTML5
- CSS3
- JavaScript
